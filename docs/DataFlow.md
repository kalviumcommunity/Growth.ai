# Data Flow

## Workflow

```
User Activity Logs
        │
        ▼
SQL Database
        │
        ▼
SQL Extraction
        │
        ▼
Pandas DataFrame
        │
        ▼
Data Cleaning
        │
        ▼
Data Transformation
        │
        ▼
NumPy Calculations
        │
        ▼
Business KPIs
        │
        ▼
Dashboard
        │
        ▼
Product Insights
```

---

## Processing Steps

### Step 1

Extract historical data using SQL.

### Step 2

Load data into Pandas.

### Step 3

Clean missing values.

### Step 4

Remove duplicate records.

### Step 5

Transform timestamps.

### Step 6

Compute KPIs using NumPy.

### Step 7

Generate charts and reports.
