# User Behavior Analytics & Subscription Conversion Dashboard

## Overview

This project analyzes six months of SaaS user activity data to discover behavioral patterns that influence free trial users to upgrade to paid subscriptions.

The solution leverages SQL, Pandas, and NumPy to process historical datasets, compute key business metrics, and generate dashboards that help product managers make data-driven decisions.

---

## Problem Statement

A SaaS company has six months of free trial user activity, feature usage logs, and subscription conversion data, yet product managers still rely on intuition because no workflow connects user behavior patterns to successful upgrades.

---

## Objectives

- Analyze user activity
- Measure feature adoption
- Calculate conversion metrics
- Compare trial vs paid users
- Generate business insights
- Support product decision-making

---

## Tech Stack

- Python
- SQL
- Pandas
- NumPy
- Matplotlib (optional)
